## 📌학교 게시판, 입학처url 크롤링
- 학교 게시판의 link와 tag를 통해 크롤링하는 매크로와 네이버에 검색하여 자동으로 (매번 바뀌는) 입학처 링크를 취득하는 코드입니다.
- 고입 경쟁률 공시, 추가 합격자 발표, 교육부 발표자료, 변환 표준 점수를 실시간으로 편하게 조회하기 위해 만든 매크로입니다.
- 주요 성과 : 2 ~ 3명으로 동시에 20 ~ 30건 커버가능
