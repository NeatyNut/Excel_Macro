## 📌인원 조사 관련
- 일단위로 진행되는 인원 조사 업무
- 실시간 집계가 아니기에 나타날 수 있는 오류 점검(마감기준 실제 전일 인원 vs 어제 보고된 전일 인원)
- 인트라넷을 통해 공시되는 인원표와 보고해야 하는 인원표 모양 다름, 매일 인원표 여러 모양으로 보관해야 함
- 주요성과 : 30분 이상 소요 가능 → 최대 5분 소요
- 당연하게도 전처리 관련 코드만 포함시켰습니다.
